## About

**PvZ: Overgrown** is tower defense game unique ALT worlds system (alternate plant/world variants).

## Features

- 5 worlds × 10 levels each + ALT variants
- 35+ plants with unique behaviors 
- Multiple zombie types with special abilities and armor
- Conveyor belt levels
- Debug console with god mode, wave skip, and stress tests
- Save system with star ratings and progression

## Debug Controls

| Key | Action |
|-----|--------|
| `F1` | Toggle debug overlay |
| `F2` | God mode |
| `F3` | Free plants |
| `F4` | Skip wave |
| `F5` | Kill all zombies |
| `F6` | Add sun |
| `` ` `` | Open command console |
